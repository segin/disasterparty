# Disaster Party - Comprehensive Project Roadmap

## 1. Recently Completed (v0.6.0)

*   **Tool Calling:** Full implementation for OpenAI, Gemini, and Anthropic.
*   **Thinking Tokens:** Support for internal model reasoning blocks.
*   **Detailed Streaming:** Unified cross-provider detailed event stream.
*   **Consumer Protection:** Integrated safety chunking for streaming tokens.
*   **Modularization:** Codebase refactored into domain-specific modules.
*   **Image Generation:** Support for DALL-E and Imagen.

## 2. Near-Term Goals

*   **Expanded Test Coverage:** More edge cases for tool calling and thinking.
*   **Advanced File Handling:** Further testing of diverse MIME types.
*   **Documentation Site:** Potential Doxygen or Sphinx site.

## 3. Long-Term Goals

*   **Speech-to-Text / Text-to-Speech:** Transcription and voice generation.
*   **Real-time Audio:** WebSocket-based low-latency interaction.
*   **Video Generation:** Support for Sora/Veo APIs.
